# GhostNet Pro 

GhostNet Pro is an AI-powered scam and fraud detection platform that analyzes
messages and URLs to identify phishing, UPI scams, job scams, and more.

## Features
- Google Authentication (Firebase Auth)
- AI-based scam detection (Gemini AI)
- URL & message scanning
- Multilingual support (Indian languages)
- Real-time scan history (Firestore)
- Dark / Light mode
- Mobile-optimized UI

##  Tech Stack
- HTML, CSS, JavaScript
- Firebase (Auth, Firestore, Hosting)
- Google Gemini AI
